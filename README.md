# cloud-webapp 

    云函数爬虫豆瓣网站，云函数登陆、路由tcb-router模块爬虫理由设计，没有什么了、、、

## 用到的模块
- request-promise 请求数据
- cheerio 获取Dom节点
- tcb-router 云函数路由
- doubanbook 解密数据
